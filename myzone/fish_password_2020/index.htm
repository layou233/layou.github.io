本项目提供JSON接口

如果在本网址后面加上

?json=

则会自动输出当日暗号的JSON

输出的内容类型是由字符串组成的列表

json值即上方直达入口使用的日期码，即当日的月份和日期连接起来的三位数字，如3月27日，json的值应为327

e.g.

window.location.href="https://layou233.github.io/myzone/fish_password_2020/?json=327"

Output=

{ dayOfFishPw = "327", fishPassword = ["番茄"] }
